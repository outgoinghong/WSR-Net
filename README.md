# WSR-Net
# Wavelet-Driven Spectral-Spatial Reciprocal Network for Hyperspectral Unmixing With Spectral Variability
The code in this toolbox implements the "Wavelet-Driven Spectral-Spatial Reciprocal Network for Hyperspectral Unmixing With Spectral Variability".

# System-specific notes
The code was tested in the environment of `Python 3.8`, `torch 1.9.1` and `CUDA 11.3`.
